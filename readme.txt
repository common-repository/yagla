Плагин для работы с сервисом yagla.ru

== Description ==

Плагин добавляет на сайт код от сервиса yagla.ru
Дополнительно можно включить отправку событий для отслеживания целей в сервисе yagla.ru для других плагинов.
Поддерживается: 
 - Contact Form 7


== Installation ==

1. Найдите плагин yagla на странице <strong>Добавить плагины</strong> и установите его. 
2. Активируйте плагин через меню <strong>Плагины</strong>.
3. Зайдите на страницу настройки плагина Настройки > Yagla.
4. Укажите код из вашего проекта на сайте yagla.ru и нажмите "Установить код скрипта" или укажите ваш логин и пароль от сервиса yagla.ru,чтобы плагин сам получил его из вашего личного кабинета, и нажмите "Получить код скрипта и установить" . 
5. Готово. Скрипт установлен, можно пользоваться.
